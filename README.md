# 4ITB PRG - Vlastní Blockchain s REST API

Tento projekt je jednoduchá demonstrace fungování technologie blockchain. Obsahuje základní logiku pro vytváření bloků, přidávání transakcí a těžbu (Proof of Work). Komunikace s blockchainem probíhá přes REST API vytvořené v Python frameworku Flask.

## 🛠 Použité technologie
- **Python 3.x** - hlavní programovací jazyk
- **Flask** - odlehčený framework pro vytvoření REST API
- **Hashlib** - standardní Python knihovna pro SHA-256 hashování

## 🚀 Jak projekt spustit

1. Ujistěte se, že máte nainstalovaný Python.
2. Nainstalujte potřebnou knihovnu Flask pomocí terminálu:

   pip install Flask
